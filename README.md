# Customer Churn Analysis Dashboard using Power BI

## Project Overview
This project focuses on analyzing customer churn data using Power BI to identify factors influencing customer attrition and provide actionable business insights. The dashboard helps organizations understand customer behavior, detect high-risk customers, and improve retention strategies.

## Objectives
- Analyze customer churn trends and patterns.
- Identify customers with a high risk of churn.
- Understand the key reasons behind customer attrition.
- Examine demographic, behavioral, and geographic factors affecting churn.
- Support data-driven customer retention decisions.

## Dashboard Pages

### Home
Project introduction and navigation.

### Summary
Key performance indicators (KPIs) and overall churn statistics.

### Analysis
Detailed analysis of customer demographics and churn metrics.

### Behaviour
Customer behavior patterns related to churn.

### Geography
Geographical distribution of customers and churn rates.

### Reasons
Analysis of major factors contributing to customer churn.

### Insights
Key findings and business recommendations derived from the analysis.

## Key Features

- High-risk customer identification
- Churn reason analysis
- Business insights and recommendations

## Tools & Technologies
- Power BI Desktop

## Dataset
The project uses the Telco Customer Churn dataset containing customer demographic information, subscription details, and churn status.

## Project Files
- `customerchurn.pbix` – Power BI dashboard file
- `Telco_customer_churn.xlsx` – Dataset used for analysis
- `customerchurn.pdf` – Exported dashboard report

## Key Insights
- Certain customer segments show significantly higher churn rates.
- Customer behavior and service usage strongly influence churn probability.
- Geographic variations impact customer retention.
- High-value customers at risk can be identified for targeted retention campaigns.
